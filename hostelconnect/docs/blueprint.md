# **App Name**: Hostel Connect

## Core Features:

- Secure Role-Based Authentication: Implement a robust login/signup system with distinct roles for Students and Employees, ensuring secure access and redirecting users to their respective dashboards upon successful authentication.
- Dynamic Mess Menu Display & Management: Provide a module for employees to effortlessly add, edit, or delete daily meal schedules (Breakfast, Lunch, Dinner) across multiple messes. Students can view the organized weekly menu with options to switch between messes.
- Service Request System: Enable students to submit service requests (e.g., house help, electrician) by category with detailed descriptions and track their status. Employees can view all pending requests, update their status, and optionally assign staff.
- Lost & Found Hub: A centralized system where users can post descriptions and images of lost or found items, including contact information. Features include a filter to toggle between 'lost' and 'found' posts for easier searching.
- Announcements & Communication Portal: Employees can post important announcements with titles, descriptions, and priority levels. Students will see these announcements prominently in their dashboard and via optional pop-up alerts.
- Anonymous Feedback Tool: Students can submit anonymous complaints without revealing their identity. An AI tool will assist administrators in summarizing and categorizing these complaints, facilitating prompt identification and resolution of sensitive issues.
- User-Friendly Dashboard Navigation: A responsive sidebar-based dashboard providing intuitive navigation across all application modules, tailored to display relevant options for each user role (Student or Employee).

## Style Guidelines:

- Primary color: A welcoming, modern blue (#227FD9), symbolizing trust and clarity. Chosen for its calm energy and excellent contrast in a light theme.
- Background color: A soft, heavily desaturated light blue (#ECF3F8), providing a clean and minimal canvas that subtly complements the primary blue.
- Accent color: A vibrant cyan-turquoise (#33E6E6), used sparingly for interactive elements like buttons and highlights to guide user attention and add a touch of freshness.
- Body and headline font: 'PT Sans' (sans-serif) for its blend of modern readability and a hint of warmth, suitable for both concise labels and longer descriptive texts within a structured dashboard environment.
- Utilize a consistent set of clean, line-style icons that visually represent each dashboard module, ensuring easy recognition and navigation.
- Emphasize a card-based layout for content organization within dashboards (e.g., mess menu items, service requests), grouped clearly by relevant categories, alongside a responsive sidebar navigation.
- Incorporate subtle, smooth hover animations for interactive elements like buttons and navigation links, and seamless transitions between different dashboard views to enhance the user experience.
